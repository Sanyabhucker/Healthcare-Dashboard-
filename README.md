Healthcare Waiting Time Dashboard

📊 Project Overview

This project was developed as part of my learning experience in data analytics using Power BI. The dashboard analyzes healthcare referral-to-treatment waiting times between 2018 and 2021.

🎯 Objectives

Analyze average and median waiting times
Compare waiting times across case types (Outpatient, Day Case, Inpatient)
Identify trends over time (2018–2021)
Examine distribution of patients across different time bands and age groups

🔍 Key Insights

A significant number of patients fall into the longer waiting time bands, especially 18+ months.
Outpatient cases make up the majority of total volume.
There is a noticeable increase in outpatient activity over time, particularly after 2019.
The difference between average and median waiting times suggests the presence of outliers and skewed data.

🛠 Tools & Skills Used

Power BI,
Data cleaning and transformation (Power Query), 
Data modeling, 
DAX (for calculated measures such as averages and medians), 
Data visualization and dashboard design
